Proyecto Big Data - Maestría en Analítica de Datos

Descripción del Proyecto

Este proyecto está construido sobre una arquitectura cliente-servidor que expone endpoints para interactuar con MongoDB. Su interfaz responsiva basada en componentes ofrece un dashboard interactivo donde los usuarios pueden navegar entre bases de datos, colecciones y documentos. Incluye un editor de consultas con autocompletado de sintaxis y un sistema de paginación que optimiza la visualización de grandes volúmenes de datos. El sistema integra mecanismos de autenticación y control de accesos, gestión de sesiones y manejo centralizado de errores para garantizar seguridad y estabilidad.
